# @hydric/gateway

The official TypeScript SDK for the hydric Gateway API.

## Installation

```bash
npm install @hydric/gateway
# or
pnpm add @hydric/gateway
```

## Development

### Prerequisites

- Node.js >= 18

### Commands

```bash
# Generate types from OpenAPI spec
pnpm generate-types

# Build (ESM + CJS)
pnpm build

# Run tests
pnpm test
```
